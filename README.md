# Cybersecurity Training Program – Yiga Elvis

This repository documents my structured cybersecurity training journey and portfolio development.

The program follows an **18-month evidence-driven roadmap** designed to build real-world defensive security skills through practical labs, professional reporting, and portfolio artifacts.

All activities are performed in **authorized lab environments only**.

## Program Overview

The training program is divided into **three progressive semesters**:

### Semester 1 – Foundations (Months 1–6)

Focus: Core cybersecurity and system fundamentals.

Topics Covered:

- Computer hardware and operating system fundamentals
- Virtualization and secure lab setup
- Networking fundamentals (IP, DNS, TCP/IP, routing)
- Linux system administration and hardening
- Python for security automation
- Web security basics (OWASP concepts)
- Security principles (CIA triad, threat modeling)
- Identity and access management
- Security monitoring fundamentals
- Incident response fundamentals
- Introduction to digital forensics
- Cloud security basics
- Security automation fundamentals

Deliverables:

- Weekly evidence packs
- Lab reports
- Security scripts
- Network diagrams
- Incident response reports

## Home Lab Environment

The training program is performed in a controlled home lab environment including:

- Kali Linux (attacker simulation & tools)
- Ubuntu Linux (system administration & hardening)
- Windows VM (endpoint telemetry and logging)
- Metasploitable / Juice Shop (training targets)
- Host-only isolated network

Tools used include:

- Wireshark
- Zeek
- Suricata
- Sysmon
- Elastic / SIEM concepts
- Python automation scripts
- Volatility
- YARA
